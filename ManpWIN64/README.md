## 🔬 Key Features

### Advanced Mathematics
- **240+ fractal types** (Mandelbrot variants, Julia sets, Newton, IFS, attractors)
- **Deep zoom** with perturbation theory (magnification > 10^100)
- **Arbitrary precision** arithmetic (MPFR, QD libraries)
- **BLA acceleration** for extreme performance

### Rendering Capabilities
- Escape-time, slope/derivative shading, distance estimation
- 24-bit true color with smooth gradients
- Fractint .map palette support
- Multithreaded engine

## 🆚 ManpWIN64 vs ManpWinUI

| Feature | ManpWIN64 (C++) | ManpWinUI (WinUI 3) |
|---------|-----------------|---------------------|
| **Fractals** | 240+ types | 2 types (Mandelbrot, Julia) |
| **Zoom** | Arbitrary precision (10^100+) | Standard precision |
| **UI** | Win32 API | Modern WinUI 3 |
| **Target** | Research, advanced users | Learning, beginners |
